# 🧹 Data Cleaning Project – Airbnb Dataset

## 📌 Project Overview

This project focuses on cleaning an Airbnb dataset to make it ready for analysis.

## 🔧 Tools Used

* Python
* Pandas
* Google Colab

## 🧠 Steps Performed

* Loaded the dataset
* Checked for missing values
* Handled missing values using mean and median
* Removed unnecessary columns
* Converted data types (boolean to numeric)
* Removed duplicate records
* Cleaned column names

## 📊 Outcome

The dataset was cleaned and prepared for further data analysis and visualization.

This project demonstrates the importance of data cleaning in data analytics workflows.
